# Golang 연습 2021-12-31~
## **Golang 문법**   

> 2022-01-03정리   
>* &는 메모리주소를 참조함(ex. b:= &a 하면 a의 메모리주소값을 참조한다.)
>* *는 살펴보기 기능(ex. *b하면 a의 메모리 주소에 들어있는 실제 값을 참조한다.)      
>* Array만드는 형식 ex.
>   * ```go   
>       names := []string{"K", "L", "P"}
>       names = append(names, "J") // List에 추가하기
>       ```
